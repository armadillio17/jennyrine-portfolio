# Jennyrine Palis Jr - Portfolio

A personal portfolio website showcasing skills, projects, and experience.

## About

This portfolio was created for Jennyrine Palis Jr to showcase their work and professional profile.

## Tech Stack

- **React** - UI library
- **TypeScript** - Type-safe JavaScript
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Icon library

## AI-Assisted Development

This portfolio was created and updated with the assistance of AI tools:

- **Bolt AI** - Initial design and development
- **Claude AI** - Design updates and enhancements

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/armadillio17/jennyrine-portfolio.git

# Navigate to the project
cd jennyrine-portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build

```bash
npm run build
```

### Deploy

```bash
npm run deploy
```
